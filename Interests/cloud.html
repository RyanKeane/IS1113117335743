Cloud Computing

What is a cloud based service:


A cloud based service is a service of storage that allows a client to manage, save and access this information from any device connected to the internet. It is quite a versitile service as it alows a client to interact with there data from anywhere in a much more secure and safe alternative to traditional hard drive services.

Different providers for cloud based services:


  • Microsoft Azure
  • Google Cloud Services
  • Amazon EC2
  • Dropbox
  • Cloud9
  • Apple i Cloud Services

What is cloud computings place in business and why its so interesting:


  1. Flexability: Cloud computing allows firms to work from anyware. Which means employees can benefit from a more flexable work life balance
  2. Security: the loss of sensitive information can be catastrofic for a company Cloud computing increases security as it alows consant access to this data no matter what happenes to a particular device.
  3. Incresaed Colaberatin In Group Projects: The fact that each member of a particular team can access the group data from any location allows work to be conducted even if the group is not there in person
  4. Automatic Software Updates: Servers are off premesis and so are out of sight and out of mind software companies take care of everything including regular updates.
  5. Environmentally Friendly When your cloud needs fluctuate, your server capacity scales up and down to fit. So you only use the energy you need and you don't leave oversized carbon footprints.
  6. Highly Available:There are many different types and services available so so finding the right service is verry easy.
